# 欢迎来到我的博客！

这是我第一次使用 Hugo 搭建博客，选择了 PaperMod 主题。

## 为什么选择 Hugo？

- **速度快**：静态网站生成，访问速度极快
- **安全性高**：没有数据库，几乎零维护
- **易于部署**：可以部署到 GitHub Pages
